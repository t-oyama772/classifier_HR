# classifier_HR
***
***

当リポジトリは退職予測の実行環境です。

　　
## Directory
***
```
classifier_HR/
        ┣ bin/        # 実行ファイルの配置場所
        ┣ data/       # 退職データの配置場所
        ┣ lib/        # ライブラリの配置場所
        ┣ log/        # ログ出力先
        ┗ model/      # モデル出力先
```

　　
## Run
***
```
cd classifier_HR/bin
python classifier_HR.py
```
